UCS(Graph, Start, Goal)

Create PriorityQueue PQ
Insert (0, Start) into PQ
Create Visited Set

While PQ is not empty do
    (Cost, Node) ← Remove node with minimum cost

    If Node = Goal then
        Print Cost
        Stop
    End If

    If Node is not Visited then
        Mark Node as Visited

        For each (Neighbor, Weight) of Node do
            If Neighbor is not Visited then
                Insert (Cost + Weight, Neighbor) into PQ
            End If
        End For
    End If
End While
